# graphql-react-event-booking
#### graphql-react-event-booking is a project where i use GraphQL for the first time.
frontend works on React, that connects with backend through GraphQL. The backend is written in NodeJS, with the database models of MongoDB.
- authenticated users can create and delete events. they also can book and cancel bookings of other users' events.
- authentication process includes jwt and request header verification.
- everyone can see events available on the platform.
- every event includes details as title, description, price and date.
- for the user's bookings, it's possible to look into the price range chart.
